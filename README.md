# godot-8-directions-movement


### Movimentação de 8 direções com um KinematicBody2D
- Movimentos direcionais
- Cache de textura
- Looping de animação
- "State machine" simples - [Idle, Walk, Run]
- Mapa topdown com exemplo de colisão
